2. **ReferenceResolver.cs** — Create in `src/Dynamicweb.ContentSync/Serialization/`:
   ```csharp
   namespace Dynamicweb.ContentSync.Serialization;
   ```
   - Class `ReferenceResolver` with a constructor that accepts no arguments.
   - Internal dictionary `_pageGuidCache` (Dictionary<int, Guid>) for caching numeric page ID to GUID lookups.
   - Method `Guid? ResolvePageGuid(int numericId)`:
     - If numericId <= 0, return null.
     - Check cache first; if found, return cached value.
     - Call `Services.Pages.GetPage(numericId)` — if result is non-null, cache and return `page.UniqueId`.
     - If null (page not found), return null and log to Console.Error: `"[ContentSync] Warning: Could not resolve page ID {numericId} to GUID"`.
   - Method `Guid? ResolveParagraphGuid(int numericId)`:
     - If numericId <= 0, return null.
     - Call `Services.Paragraphs.GetParagraphsByPageId(...)` is not suitable. Instead, since DW10 does not expose `GetParagraph(int)` directly, store paragraph GUID lookups in a `_paragraphGuidCache` populated during traversal by the serializer.
     - Method `void RegisterParagraph(int numericId, Guid uniqueId)` — called by ContentMapper during paragraph mapping to populate the cache.
     - `ResolveParagraphGuid` checks cache only; if not found, logs warning and returns null.
   - Method `void Clear()` — clears all caches (call between serialization runs).

3. **ContentMapper.cs** — Create in `src/Dynamicweb.ContentSync/Serialization/`:
   ```csharp
   using Dynamicweb.Content;
   using Dynamicweb.ContentSync.Models;
   namespace Dynamicweb.ContentSync.Serialization;
   ```
   - Class `ContentMapper` with constructor accepting `ReferenceResolver resolver`.
   - Method `SerializedArea MapArea(Area area, List<SerializedPage> pages)`:
     - Maps `area.UniqueId` -> `AreaId`, `area.Name` -> `Name`, `area.Sort` -> `SortOrder`.
     - Sets `Pages` to the provided pages list.
   - Method `SerializedPage MapPage(Page page, List<SerializedGridRow> gridRows, List<SerializedPage> children)`:
     - Maps `page.UniqueId` -> `PageUniqueId`.
     - Maps `page.MenuText` -> `Name` AND `MenuText` (use MenuText for both — if `page.Name` exists as a property on DW Page, prefer it for `Name`; fall back to `page.MenuText` if `Name` does not exist or is null). Note: the DW10 docs show `MenuText` as the navigation label. Check if `page` has a `Name` property at compile time. If yes, use it. If not, use `MenuText`.
     - Maps `page.UrlName` -> `UrlName`, `page.Sort` -> `SortOrder`, `page.Active` -> `IsActive`.
     - Extracts Item fields: if `page.Item != null`, iterate `page.Item.Names`, for each name get `page.Item[name]`, if non-null add to `Fields` dictionary. Convert values to serialization-safe types (strings for most; preserve numeric types).
     - Sets `GridRows` to provided gridRows, `Children` to provided children.
   - Method `SerializedGridRow MapGridRow(GridRow gridRow, List<SerializedGridColumn> columns)`:
     - Maps `gridRow.UniqueId` -> `Id`, `gridRow.Sort` -> `SortOrder`.
     - Sets `Columns` to the provided columns list.
   - Method `SerializedParagraph MapParagraph(Paragraph paragraph)`:
     - Maps `paragraph.UniqueId` -> `ParagraphUniqueId`, `paragraph.Sort` -> `SortOrder`.
     - Maps `paragraph.ItemType` -> `ItemType`, `paragraph.Header` -> `Header`.
     - Register paragraph with resolver: `_resolver.RegisterParagraph(paragraph.ID, paragraph.UniqueId)`.
     - Extracts Item fields same as Page.
     - **Reference field resolution**: For known reference fields, check the paragraph properties:
       - `paragraph.MasterParagraphID`: if > 0, resolve via `_resolver.ResolveParagraphGuid()`, store as `Fields["MasterParagraphGuid"]` (GUID string). Do NOT serialize the numeric value.
       - `paragraph.GlobalRecordPageID`: if > 0, resolve via `_resolver.ResolvePageGuid()`, store as `Fields["GlobalRecordPageGuid"]` (GUID string).
     - If paragraph has `Text` property and it's non-null/non-empty, include as `Fields["Text"]`.
   - Method `List<SerializedGridColumn> BuildColumns(IEnumerable<Paragraph> paragraphs)`:
     - Group paragraphs by `GridRowColumn` property (int), ordered by column key ascending.
     - For each group: create `SerializedGridColumn` with `Id = column key`, `Width = 0` (not available from DW API), `Paragraphs = group items mapped via MapParagraph, ordered by Sort`.
     - If no paragraphs, return single column with Id=1, Width=0, empty Paragraphs.

2. Class `ContentSerializer` with constructor accepting:
   - `SyncConfiguration configuration`
   - `IContentStore? store = null` (defaults to `new FileSystemStore()` if null — allows test injection)

3. Private fields:
   - `_configuration` (SyncConfiguration)
   - `_store` (IContentStore)
   - `_referenceResolver` (ReferenceResolver — created in constructor)
   - `_mapper` (ContentMapper — created in constructor with the resolver)
   - `_predicateSet` (ContentPredicateSet — created from configuration)

4. Public method `void Serialize()`:
   - For each predicate in `_configuration.Predicates`:
     - Call `SerializePredicate(predicate)`.
   - Clear reference resolver caches between predicates: `_referenceResolver.Clear()`.

5. Private method `SerializedArea SerializePredicate(PredicateDefinition predicate)`:
   - Fetch area: `var area = Services.Areas.GetArea(predicate.AreaId)`. If null, log warning and return.
   - Find root pages for the predicate. Strategy:
     - Get all top-level pages: `Services.Pages.GetPagesByParentID(0)` filtered by `page.AreaId == predicate.AreaId`, OR use area's root page approach.
     - Better: iterate `Services.Pages.GetPagesByParentID(area.FirstActivePage?.ID ?? 0)` — actually, use `Services.Pages.GetPagesByParentID(predicate.AreaId)` if that returns area-level pages, or find the root pages by checking pages where `ParentPageId` points to area root.
     - Simplest reliable approach: `var rootPages = Services.Pages.GetPagesByParentID(0).Where(p => p.AreaId == predicate.AreaId)` — top-level pages of the area. If this doesn't work, try `Services.Pages.GetPagesByAreaID(predicate.AreaId)` and filter to those with no parent.
   - For each root page, build the content path as `"/" + page.MenuText` (matching predicate path format).
   - Filter root pages through `_predicateSet.ShouldInclude(contentPath, predicate.AreaId)`.
   - For matching root pages, call `SerializePage(page, predicate, contentPath)`.
   - Assemble into `SerializedArea` via `_mapper.MapArea(area, serializedPages)`.
   - Write to disk: `_store.WriteTree(serializedArea, _configuration.OutputDirectory)`.
   - Return the serialized area.

6. Private method `SerializedPage? SerializePage(Page page, PredicateDefinition predicate, string contentPath)`:
   - Check `_predicateSet.ShouldInclude(contentPath, predicate.AreaId)` — if false, return null.
   - Fetch grid rows: `Services.Grids.GetGridRowsByPageId(page.ID)`, ordered by Sort.
   - Fetch paragraphs: `Services.Paragraphs.GetParagraphsByPageId(page.ID)`.
   - For each grid row:
     - Filter paragraphs belonging to this grid row: `paragraphs.Where(p => p.GridRowId == gridRow.ID)`.
     - Build columns via `_mapper.BuildColumns(rowParagraphs)`.
     - Map grid row via `_mapper.MapGridRow(gridRow, columns)`.
   - Fetch child pages: `Services.Pages.GetPagesByParentID(page.ID)`, ordered by Sort.
   - For each child, build child content path as `contentPath + "/" + child.MenuText`.
   - Recursively call `SerializePage(child, predicate, childContentPath)`.
   - Filter out null results (excluded by predicate).
   - Map page via `_mapper.MapPage(page, serializedGridRows, serializedChildren)`.
   - Return the mapped page.
