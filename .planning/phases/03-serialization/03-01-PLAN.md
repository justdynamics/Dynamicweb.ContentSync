   - `ReadPage` must:
     - Read `page.yml`.
     - Read grid rows and paragraphs (existing logic).
     - Scan subdirectories for child page folders (those containing `page.yml`).
     - Recursively call `ReadPage` for each child page subfolder.
     - Return `page with { GridRows = gridRows, Children = childPages }`.
   - Update `ReadTree` to call `ReadPage` for each top-level page directory.
