**Public method `Deserialize()`:**
1. Call `_store.ReadTree(_configuration.OutputDirectory)` to get the `SerializedArea`.
2. For each predicate in `_configuration.Predicates`: call `DeserializePredicate(predicate, area)`.
3. Aggregate results from all predicates into a single `DeserializeResult`.
4. Log the summary via `Log(result.Summary)`. If `result.HasErrors`, log each error.
5. Return the `DeserializeResult`.

**Private method `DeserializePredicate(PredicateDefinition predicate, SerializedArea area)`:**
1. Resolve the target area's numeric ID: call `Services.Areas.GetArea(predicate.AreaId)`. If null, log warning and return empty result.
2. Build page GUID cache: `var allPages = Services.Pages.GetPagesByAreaID(predicate.AreaId);` then `allPages.Where(p => p.UniqueId != Guid.Empty).ToDictionary(p => p.UniqueId, p => p.ID)` stored in `ctx.PageGuidCache`.
3. Create WriteContext with `TargetAreaId = predicate.AreaId`, `ParentPageId = 0`.
4. For each root page in `area.Pages`: call `DeserializePageSafe(page, ctx)`.
5. Return a `DeserializeResult` from the ctx counters.

**Private method `DeserializePageSafe(SerializedPage dto, WriteContext ctx)`:**
Wraps DeserializePage in try/catch. On exception:
- Increment `ctx.Failed`.
- Add error message: `$"ERROR deserializing page {dto.PageUniqueId} ('{dto.MenuText}'): {ex.Message}"`.
- Add GUID to `ctx.FailedParentGuids` so children are cascade-skipped.
- Log: `$"  SKIPPED children of {dto.PageUniqueId} due to parent failure"`.

Before calling DeserializePage, check if ANY ancestor GUID is in `ctx.FailedParentGuids`. If so, increment `ctx.Skipped` and log `$"SKIPPED page {dto.PageUniqueId} ('{dto.MenuText}') — parent failed"`, then return without processing.

**Private method `DeserializePage(SerializedPage dto, WriteContext ctx)` returning `int` (resolved numeric page ID):**

INSERT path (GUID not in `ctx.PageGuidCache`):
- If `_isDryRun`: log `$"[DRY-RUN] CREATE page {dto.PageUniqueId} ('{dto.MenuText}')"` and log each field with `$"  set {key} = '{value}'"`. Increment `ctx.Created`. Return -1.
- Create `var page = new Page()`, set:
  - `page.UniqueId = dto.PageUniqueId`
  - `page.AreaId = ctx.TargetAreaId`
  - `page.ParentPageId = ctx.ParentPageId`
  - `page.MenuText = dto.MenuText`
  - `page.UrlName = dto.UrlName`
  - `page.Active = dto.IsActive`
  - `page.Sort = dto.SortOrder`
  - Do NOT set `page.ID` (leave 0 for insert path — Pitfall 4).
- Call `var saved = Services.Pages.SavePage(page)`.
- Cache: `ctx.PageGuidCache[dto.PageUniqueId] = saved.ID`.
- Apply ItemType fields in a post-save update (Pitfall 3): re-fetch page via `Services.Pages.GetPage(saved.ID)`, then if `refetched.Item != null` and `dto.Fields.Count > 0`, set fields via `refetched.Item[key] = value` for each kvp in `dto.Fields`. Handle "Text" field specially: if key is "Text", also set it on the paragraph's Text property (this only applies to paragraphs, not pages — for pages just set via Item). Then call `Services.Pages.SavePage(refetched)` again.
- Increment `ctx.Created`. Log `$"CREATED page {dto.PageUniqueId} -> ID={saved.ID}"`.
- Return `saved.ID`.

UPDATE path (GUID found in cache, `existingId`):
- If `_isDryRun`: load existing page via `Services.Pages.GetPage(existingId)`, call `LogDryRunPageUpdate(dto, existingPage)`, return `existingId`.
- Create `var page = new Page()`, set all same fields as insert path, PLUS `page.ID = existingId`.
- If existing page has `Item != null`, apply ItemType fields: full replace — for each kvp in `dto.Fields`, set `page.Item[kvp.Key] = kvp.Value`. For fields in existing Item that are NOT in `dto.Fields`, attempt to clear by setting `page.Item[fieldName] = null`.
- Call `Services.Pages.SavePage(page)`.
- Increment `ctx.Updated`. Log `$"UPDATED page {dto.PageUniqueId} (ID={existingId})"`.
- Return `existingId`.

After page is written (both paths), process grid rows and children:
- Build grid row GUID cache: `Services.Grids.GetGridRowsByPageId(resolvedPageId).Where(gr => gr.UniqueId != Guid.Empty).ToDictionary(gr => gr.UniqueId, gr => gr.ID)`.
- For each grid row in `dto.GridRows`: call `DeserializeGridRowSafe(row, resolvedPageId, gridRowCache, ctx)`.
- For each child in `dto.Children`: set `ctx.ParentPageId = resolvedPageId`, call `DeserializePageSafe(child, ctx)`.

**Private method `DeserializeGridRowSafe(SerializedGridRow dto, int pageId, Dictionary<Guid, int> gridRowCache, WriteContext ctx)`:**
Try/catch wrapper. On error: increment `ctx.Failed`, add error, log. On success:

INSERT path (GUID not in gridRowCache):
- If `_isDryRun`: log CREATE, increment `ctx.Created`, return.
- `var row = new GridRow(pageId)`, set `row.UniqueId = dto.Id`, `row.Sort = dto.SortOrder`. Do NOT set `row.ID`.
- `Services.Grids.SaveGridRow(row)` returns `bool`.
- Re-query to get assigned ID (Pitfall 1): `var saved = Services.Grids.GetGridRowsByPageId(pageId).FirstOrDefault(gr => gr.UniqueId == dto.Id)`. If null, throw `InvalidOperationException($"Could not find inserted grid row with GUID {dto.Id}")`.
- `var newGridRowId = saved.ID`. Increment `ctx.Created`. Log.

UPDATE path (GUID found, `existingGridRowId`):
- If `_isDryRun`: log UPDATE with sort order diff if changed. Increment `ctx.Updated` or `ctx.Skipped`. Return.
- `var row = new GridRow(pageId)`, set `row.ID = existingGridRowId`, `row.UniqueId = dto.Id`, `row.Sort = dto.SortOrder`.
- `Services.Grids.SaveGridRow(row)`. Increment `ctx.Updated`. Log.

After grid row is written, process paragraphs:
- Build paragraph GUID cache: `Services.Paragraphs.GetParagraphsByPageId(pageId).Where(p => p.UniqueId != Guid.Empty).ToDictionary(p => p.UniqueId, p => p.ID)`.
- For each column in `dto.Columns`, for each paragraph in `column.Paragraphs`: call `DeserializeParagraphSafe(para, pageId, resolvedGridRowId, column.Id, paragraphCache, ctx)`.

**Private method `DeserializeParagraphSafe(SerializedParagraph dto, int pageId, int gridRowId, int columnId, Dictionary<Guid, int> paragraphCache, WriteContext ctx)`:**
Try/catch wrapper. On error: increment `ctx.Failed`, add error, log.

INSERT path:
- If `_isDryRun`: log CREATE with all fields. Increment `ctx.Created`. Return.
- `var para = new Paragraph()`, set:
  - `para.UniqueId = dto.ParagraphUniqueId`
  - `para.PageID = pageId`
  - `para.GridRowId = gridRowId`
  - `para.GridRowColumn = columnId`
  - `para.Sort = dto.SortOrder`
  - `para.Header = dto.Header`
  - `para.ItemType = dto.ItemType`
  - Do NOT set `para.ID` (insert path).
- `Services.Paragraphs.SaveParagraph(para)`.
- Re-query to get assigned ID: `var saved = Services.Paragraphs.GetParagraphsByPageId(pageId).FirstOrDefault(p => p.UniqueId == dto.ParagraphUniqueId)`.
- If saved != null and `dto.Fields.Count > 0`: apply ItemType fields on the re-fetched paragraph. For "Text" key: set `saved.Text = value?.ToString()`. For other keys: `saved.Item[key] = value`. Then `Services.Paragraphs.SaveParagraph(saved)`.
- Increment `ctx.Created`. Log.

UPDATE path (existingParagraphId):
- If `_isDryRun`: load existing paragraph, call `LogDryRunParagraphUpdate(dto, existing)`. Return.
- Load existing: `var existing = Services.Paragraphs.GetParagraphsByPageId(pageId).FirstOrDefault(p => p.ID == existingParagraphId)`.
- Set all properties: `existing.UniqueId = dto.ParagraphUniqueId`, `existing.GridRowId = gridRowId`, `existing.GridRowColumn = columnId`, `existing.Sort = dto.SortOrder`, `existing.Header = dto.Header`, `existing.ItemType = dto.ItemType`.
- Apply ItemType fields with full replace: for each field in existing `Item.Names.ToList()` that's NOT in `dto.Fields`, set `existing.Item[fieldName] = null`. Then for each kvp in `dto.Fields`: if key == "Text", set `existing.Text = value?.ToString()`; else `existing.Item[key] = value`.
- `Services.Paragraphs.SaveParagraph(existing)`. Increment `ctx.Updated`. Log.

**Private dry-run diff methods:**

`LogDryRunPageUpdate(SerializedPage dto, Page existing)`:
- Compare: MenuText, UrlName, IsActive/Active, SortOrder/Sort.
- Compare ItemType fields: for each kvp in dto.Fields, compare `existing.Item?[key]?.ToString()` vs `value?.ToString()`.
- If no diffs: log `$"[DRY-RUN] SKIP {dto.PageUniqueId} (unchanged)"`, increment `ctx.Skipped`.
- If diffs: log `$"[DRY-RUN] UPDATE {dto.PageUniqueId}:\n  "` + diffs joined by `"\n  "`. Increment `ctx.Updated`.

`LogDryRunParagraphUpdate(SerializedParagraph dto, Paragraph existing)`:
- Compare: SortOrder/Sort, Header, ItemType, and each field in dto.Fields vs existing.Item.
- Same SKIP/UPDATE logging pattern.
